# Ajustes de navegação e jornada

- O cadastro não carrega mais o modo escuro nem exibe o botão de tema.
- Os cabeçalhos públicos e administrativos agora usam fundos sólidos, sem transparência ou desfoque.
- O selo visual "novo" foi removido do item New User.
- A jornada New User é atualizada automaticamente conforme o usuário:
  1. entra em uma conta;
  2. acessa categorias;
  3. abre os detalhes de um curso;
  4. acessa o feed.
- A seção "Selecionado para você" usa cursos reais e direciona aos detalhes correspondentes.
- Os cartões de Programas Estratégicos direcionam para páginas oficiais do Sebrae.
