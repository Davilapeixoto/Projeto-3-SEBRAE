# Ferramenta KI

## Sobre o Projeto

A **Ferramenta KI** é um projeto desenvolvido em parceria com o **SEBRAE**, com o objetivo de analisar o comportamento real dos usuários em uma plataforma digital para identificar sinais de satisfação, engajamento ou dificuldade sem perguntar diretamente a eles.

A ferramenta utiliza dados como tempo de tela, taxas de clique, navegação e padrões de uso para apoiar uma avaliação mais natural da experiência do usuário.

## Problema e Solução

Muitas plataformas dependem de formulários ou avaliações diretas para medir a satisfação dos usuários. No entanto, esses métodos podem ter baixa adesão e nem sempre representam o comportamento real durante o uso da plataforma.

A **Ferramenta KI** busca solucionar essa dor analisando indicadores comportamentais, como:

- Tempo de permanência nas telas;
- Taxas de clique;
- Fluxo de navegação;
- Categorias mais acessadas;
- Pontos de abandono;
- Interações com conteúdos.

Com esses dados, a solução permite identificar padrões de uso e possíveis pontos de melhoria na experiência digital.

## Tecnologias Utilizadas

- **Backend:** Java, Spring Boot e API REST;
- **Frontend:** React, JavaScript/TypeScript, HTML e CSS;
- **Design e documentação:** Figma, fluxo do usuário, jornada do usuário, protótipos e diagrama de classes.

## Equipe

O projeto foi desenvolvido por uma equipe composta por **4 designers** e **6 estudantes de Ciência da Computação**, integrando visão de experiência do usuário, modelagem, frontend e backend.

## Considerações Finais

A **Ferramenta KI** propõe uma forma menos invasiva e mais orientada a dados para compreender a satisfação dos usuários, auxiliando o SEBRAE na melhoria contínua da experiência digital.
